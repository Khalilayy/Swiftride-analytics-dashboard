# SwiftRide Analytics Dashboard
Data analytics dashboard built using python and streamlit to analyze ride sharing data
An interactive data dashboard built using Streamlit to analyze and visualize car rental / ride-related data.

---

## 📊 Features

- Interactive UI built with Streamlit
- Dynamic data visualization
- Data generation using custom scripts
- Clean and responsive dashboard layout
- Easy-to-use interface

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/usernam/Swiftride-analytics-dashboard.git
cd Swiftride-analytics-dashboard

2. Create virtual environment 
python -m venv .venv
.venv\Scripts\activate

3. Install dependencies
pip install streamlit pandas numpy matplotlib

▶️ Run the App
streamlit run app.py
